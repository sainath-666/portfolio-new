# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this portfolio, please email sai65265@gmail.com instead of using the issue tracker.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |
| Older   | ⚠️        |

## Security Measures

- All dependencies are kept up to date
- Input validation is implemented
- HTTPS is enforced on production
- Security headers are configured on Vercel
